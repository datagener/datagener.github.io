# Local File

This is for all connectors to local file system: **CSV, Avro, JSON, ORC, Parquet**.

## Parameters 

Required Parameters:

- **Local File Path**
- **Local File Name**: Used as a prefix to name files generated (a number on 10 digits and extension will be added)
