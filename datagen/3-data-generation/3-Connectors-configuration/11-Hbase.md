# HBase



## Parameters 

Required Parameters:

- **HBase Table Name**
- **HBase Namespace**
- **HBase Primary Key**

Optional Parameters:

- **HBase Column Families Mapping**:  Mapping of column family to columns in that format: cf:col1,col2,col3;cf2:col4 . (Default is cq)
