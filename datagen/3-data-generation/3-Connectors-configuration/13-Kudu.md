# Kudu



## Parameters 

Required Parameters:

- **Kudu Table Name**

Optional Parameters:

- **Kudu Number of Buckets**
- **Kudu Buffer Size**
- **Kudu Flush Type**
- **Kudu Replicas**
- **Kudu Hash Keys**: To set hash keys
- **Kudu Range Keys**: To set Range Keys
- **Kudu Primary Keys**
